This chapter covers major changes to existing APIs, and deprecated and removed APIs.

4.1. Major Changes

This maintenance release is binary-compatible with OpenDJ LDAP SDK 2.6.0. It does not introduce major changes.

4.2. Deprecation

Deprecated APIs are listed in the API Specification, on the Deprecated list page.

Deprecated APIs are likely to be removed in a future release.

4.3. Removals

Nothing has been removed from the public APIs in this release.
