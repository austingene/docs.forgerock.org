OpenDJ - File Count Log Retention Policy

File Count Log Retention Policy

Retention policy based on the number of rotated log files on disk.

Parent Component

The File Count Log Retention Policy component inherits from the Log Retention Policy

Properties

Basic Properties

number-of-files

Description
Specifies the number of archived log files to retain before the oldest ones are cleaned.
Default Value
None
Allowed Values
An integer value. Lower value is 1.
