Chapter 3. Updating & Installing OpenDJ

ForgeRock recommends that you update OpenDJ 2.6.0 installations to this release. If you are installing OpenDJ for the first time, you can use the same installation instructions as for 2.6.0.

Procedure 3.1. To Update OpenDJ Software

  1. Download and unpack OpenDJ 2.6.2 software.

    Find a link to the OpenDJ download page from http://forgerock.com/download-stack/.

  2. Follow the instructions in the chapter on Upgrading to OpenDJ 2.6.0 in the Installation Guide.

    When upgrading from OpenDJ 2.5.0-Xpress1, you no longer need to rebuild the ds-sync-hist index as a separate step.

Procedure 3.2. To Install OpenDJ Software

  1. Download and unpack OpenDJ 2.6.2 software.

    Find a link to the OpenDJ download page from http://forgerock.com/download-stack/.

  2. Follow the instructions in the OpenDJ 2.6.0 Installation Guide.
